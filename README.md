## 🌟 Destaque: Meu Primeiro Grande Projeto - Gerenciador de Tarefas!

Este é o meu xodó! O **Gerenciador de Tarefas** é um sistema completo para organizar suas atividades diárias, desde as mais simples até as mais complexas. Foi aqui que pude ver a teoria se transformar em prática, construindo uma aplicação do zero ao deploy (local, por enquanto 😉).

### 💡 O que é o Projeto?

É uma aplicação web robusta, desenvolvida com **Spring Boot** no backend e uma interface dinâmica com **HTML, CSS e JavaScript (Thymeleaf)**. Ele permite que você crie, visualize, atualize e exclua tarefas, além de gerenciá-las por prioridade e status de conclusão.

### ✨ Principais Funcionalidades

*   **Criação de Tarefas:** Adicione novas tarefas com título, descrição e prioridade.
*   **Visualização Detalhada:** Veja todas as suas tarefas em uma lista organizada.
*   **Marcação de Conclusão:** Marque tarefas como concluídas ou reabra-as facilmente.
*   **Exclusão de Tarefas:** Remova tarefas que não são mais necessárias.
*   **Filtros Inteligentes:** Filtre tarefas por status (todas, pendentes, concluídas).
*   **Estatísticas em Tempo Real:** Acompanhe o total de tarefas, concluídas e pendentes.
*   **Priorização:** Defina prioridades (Baixa, Média, Alta) para suas tarefas.

### 🚀 Como Rodar Localmente

Quer testar o Gerenciador de Tarefas na sua máquina? Siga estes passos simples:

1.  **Pré-requisitos:**
    *   Java Development Kit (JDK) 17 ou superior.
    *   Maven (gerenciador de dependências).
    *   Um IDE de sua preferência (IntelliJ IDEA, VS Code, Eclipse).

2.  **Clone o Repositório:**
    ```bash
    git clone https://github.com/seu-usuario/seu-repositorio.git
    cd seu-repositorio/MultipleFiles # Caminho para a pasta raiz do projeto Spring Boot
    ```

3.  **Compile o Projeto com Maven:**
    ```bash
    ./mvnw clean install
    ```
    *Se estiver no Windows, use `mvnw.cmd clean install`.*

4.  **Execute a Aplicação:**
    ```bash
    ./mvnw spring-boot:run
    ```
    *Se estiver no Windows, use `mvnw.cmd spring-boot:run`.*

5.  **Acesse no Navegador:**
    Abra seu navegador e acesse: `http://localhost:8080`

Pronto! O Gerenciador de Tarefas estará rodando localmente.

### 💻 Tecnologias Utilizadas no Projeto

*   **Backend:**
    *   **Java 17:** Linguagem de programação principal.
    *   **Spring Boot:** Framework para construção rápida de aplicações Java.
    *   **Spring Data JPA:** Para persistência de dados e interação com o banco de dados.
    *   **Spring Web:** Para construção de APIs REST e controladores web.
    *   **H2 Database:** Banco de dados em memória para desenvolvimento e testes.
*   **Frontend:**
    *   **HTML5:** Estrutura das páginas web.
    *   **CSS3:** Estilização e design responsivo (com Bootstrap).
    *   **JavaScript:** Lógica interativa e requisições assíncronas (AJAX).
    *   **Thymeleaf:** Motor de template para integração do backend com o frontend.
    *   **Bootstrap 5:** Framework CSS para um design moderno e responsivo.
    *   **Font Awesome:** Biblioteca de ícones.
